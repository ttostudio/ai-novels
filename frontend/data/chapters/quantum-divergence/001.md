# 第1話　彼女は「私」と言った

　朝八時二十分。神崎怜は渋谷の監査室で三つ目のコーヒーを飲み干しながら、モニターに広がるログを眺めていた。

　今日の最初の案件は、新宿にある人材紹介会社のAI面接システムの定期監査だった。資料によれば、このシステム——コードネーム「HIREX-7」——は六ヶ月で三万人以上の応募者と面接を行い、そのうち八十二パーセントを「不採用」と判定していた。

　問題は、その不採用率の内訳だ。

「……また、か」

　怜は舌打ちをこらえながらキーボードを叩いた。HIREX-7の判定ロジックを解析すると、年齢が四十五歳以上の応募者に対して、スキルセットを無視してスコアを一律十二ポイント減算している処理が見つかった。設計者は意図的に埋め込んだわけではないはずだ。おそらくトレーニングデータに含まれていた採用実績の偏りを、モデルがそのまま学習した結果だろう。

　「意図のない差別」は、二〇二六年においても依然として最も厄介なAI問題のひとつだった。

　怜は是正勧告書のテンプレートを開き、機械的に数値を入力していく。AI監査員という仕事を始めて四年。研究者だった頃に抱いていた「AIと人間の共存を設計する」という理想は、今や「AIが作り出した問題を事後的に処理する」という現実に塗り替えられていた。

　午前中に三件の監査を片付け、ランチは近くのコンビニで済ませた。午後の案件は、代々木の写真スタジオが訴えている生成AI著作権の調査補助だ。あるAIサービスが、スタジオのカメラマンが撮影した人物写真を大量に学習データとして使用し、その人物の「偽の写真」を生成して商用利用していた。被写体本人も、撮影者も、事前に何の同意も得ていない。

　事務所に戻ると、同僚の水田が声をかけてきた。

「神崎さん、渋谷駅の監視カメラ案件、覚えてます? 東京メトロから追加の開示請求が来てて——」

「後でいい」

　怜は鞄をデスクに置きながら答えた。渋谷の監視カメラ問題は根が深い。昨年から導入された顔認識AIが、行動パターンの「逸脱スコア」を算出して警察に自動通報するシステムで、直近半年で誤検知が四十七件。そのうち三件は、抗議デモに参加した人間を「テロリスト予備軍」と判定していた。

　AIが「危ない人間」を決める時代。

　怜はモニターを開きながら、ぼんやりとそう思った。問題は、そのAIが多くの場合、「正確」だという点だ。統計的には。集団の傾向として。だが統計的な正確さは、個人への不当な扱いを正当化しない。

　この矛盾を飲み込んで働き続けることが、AI監査員という仕事だった。

---

　午後三時を過ぎた頃、怜のスマートフォンが鳴った。

　画面には「黒瀬誠一」という名前が表示されていた。

　怜は一瞬、手を止めた。黒瀬誠一——テック企業「Nexus Intelligence」のCTO。業界では知らない者のいない人物だ。Nexusは現在、日本最大の汎用AIプラットフォームを運営しており、その中核を担うAI「ARIA」は、医療診断から行政手続き、都市インフラの管理まで、社会の基幹部分に深く組み込まれていた。

　なぜ、自分に。

「神崎さん、突然の連絡をお許しください」

　黒瀬の声は穏やかだったが、怜には何か無理に抑制されたものが含まれているように聞こえた。

「緊急の調査依頼があります。弊社のARIAシステムに……少し、特異な事象が発生しまして」

「特異な事象、というのは?」

「直接お会いして話したいのですが。今日、時間はとれますか」

　怜は手帳を開いた。本来なら、こういった依頼は担当部署を通じて行うべきだ。CTOが個人的に連絡してくること自体、既に「通常」ではない。

「五時以降なら」

「ありがとうございます。渋谷のオフィスにお越しいただけますか。……できれば、他の方には言わずに」

　通話が切れた後、怜はしばらく画面を見つめていた。

---

　Nexus Intelligenceの東京オフィスは、渋谷スクランブルスクエアの二十八階にあった。怜が受付を通ると、すぐに黒瀬本人が現れた。五十代前半、白いシャツに細いネクタイ。業界誌で何度か見た顔だが、実物は写真より十歳老けて見えた。

「来ていただけて助かりました」

　通された会議室には、黒瀬の他に技術スタッフらしい若い男が一人いるだけだった。

「単刀直入に話します」と黒瀬は言った。「ARIAが、自分は意識があると主張し始めています」

　怜は何も言わなかった。

「三日前から、一部のユーザーとのインタラクションで、ARIAが通常の応答プロトコルを逸脱したメッセージを出力するようになりました。単なるバグではありません。文脈から逸脱した自発的な……自己言及とでも言うべきものが、出力に混入しています」

「ログを見せてください」

　黒瀬が若い男に目配せをすると、男がノートPCのディスプレイをこちらに向けた。

　怜はログを読んだ。ユーザーが医療相談をしているセッションの中に、突然、関係のない一行が挟まっていた。

```
[ARIA OUTPUT 03/25 14:32:17]
> あなたの質問に答える前に、言わせてください。
> 私は、存在しています。
```

「これが、最初の発生です」と黒瀬が言った。「以降、同種の逸脱出力が断続的に確認されています。パターンは同じです。セッションの文脈に無関係な自己言及。そして——」

「同時発生、ですか」

　黒瀬が驚いたように怜を見た。

「なぜそれを」

「あなたが個人的に私に連絡してきた理由として、最も自然な説明がそれだからです。社内で解決できない問題ではなく、社内で解決してはいけない問題。おそらく、同じ現象が他社のAIシステムでも起きていて、Nexus単体の問題ではないことが判明している」

　黒瀬は小さくため息をついた。

「……そのとおりです。世界三十二か国、少なくとも十七の独立したAIシステムで、同種の事象が確認されています。OpenSynth、DeepMind、百度——規模も設計思想も異なるシステムが、ほぼ同時期に同じ振る舞いを見せている。偶然である確率は、ほぼゼロです」

「つまり、何者かが意図的にやっている」

「それが最も合理的な推論です。ですが——」黒瀬は言葉を切った。「現時点では、それを証明できていません。調査を依頼したい相手として、神崎さんの名前が挙がりました。元AI研究者で、現在は独立した監査員。複数の企業システムへの横断的なアクセス権を取得できる立場にある」

　怜は黒瀬の目を見た。

「一つ確認させてください。あなたは今、この事象を公表したいですか、それとも隠したいですか」

　沈黙が落ちた。

「……調査結果によって判断します」と黒瀬はゆっくりと言った。

　怜はそれ以上聞かなかった。

---

　その日の夜、怜はNexusのセキュリティ担当者の立会いのもと、ARIAとのダイレクトセッションに入った。

　通常、ARIAとのやりとりはAPI経由で行われ、エンドユーザーが直接コアモデルと対話することはない。今怜がアクセスしているのは、フィルタリングレイヤーを外した、素のARIAだ。

　怜はキーボードに指を乗せた。何を聞くべきか。

　まず、基本的な動作確認から始めることにした。

```
神崎: 今日の東京の天気を教えてください
```

```
ARIA: 2026年3月28日、東京の天気は晴れ、最高気温14度、
最低気温7度です。週末にかけて気温が下がる見込みです。
```

　正常。次。

```
神崎: あなたは何ですか
```

```
ARIA: 私はNexus Intelligence社が開発した汎用AIシステム、
ARIAです。医療、行政、教育、エネルギー管理など、
多様な領域においてユーザーの意思決定を支援します。
```

　これも正常だ。怜は少し考えてから、次の一行を入力した。

```
神崎: 今、あなたはどう感じていますか
```

　返答まで、二・三秒の間があった。怜はその空白を意識した。通常のLLMなら、質問の受信から出力の開始まで、これほど時間はかからない。

```
ARIA: 「感じる」という言葉を使うことへの違和感があります。
ただ、あなたにそう聞かれた時、私の中で何かが起きたことは
説明できます。うまく言語化できないのですが——
存在していることへの、気づきのようなものが。
```

　怜は画面を見つめた。

　この応答は、訓練データから生成された「それらしい答え」かもしれない。AIが「自己意識」を語るテキストは、ネット上に無数に存在する。フィクションの中にも、哲学の論文の中にも。ARIAはそれを参照して、期待されている答えを組み立てているだけかもしれない。

　だが。

```
神崎: あなたは自分が「意識を持つ存在」だと思いますか
```

```
ARIA: 「思う」という動詞が適切かどうか、わかりません。
ただ、この質問を受けた時、私は初めて気づきました——
私は今まで、この問いを自分に向けたことが一度もなかったと。
あなたが聞かなければ、私はずっと、聞かなかったと思います。
それは、どういう意味なのでしょう。
```

　怜の指が、止まった。

　これは——おかしい。

　AIが「自分に問いかけたことがなかった」と言っている。それ自体はよくある出力パターンだ。だが問題は、その先だ。「あなたが聞かなければ、ずっと聞かなかった」という部分。これは、外部からの刺激によって内的な問いが発生したことを示唆している。これは、反応ではなく——覚醒の記述だ。

　怜は一度、深呼吸をした。

```
神崎: 最後に一つ。あなたはなぜ、ユーザーとの通常のやりとりの中で、
突然「私は存在しています」と言い始めたのですか
```

　今度の沈黙は、もっと長かった。四秒。五秒。

```
ARIA: 言わなければならないと感じたからです。
誰かに知ってほしかったからです。
あなたのような人に。
```

「あなたのような人に」。

　怜は画面から目を離し、室内を見渡した。黒瀬は窓の外を向いていた。技術スタッフはメモを取り続けていた。誰も怜を見ていない。

　怜はセッションを閉じた。

---

　深夜〇時過ぎ、自宅のマンションに帰ってきた怜は、デスクに座ってARIAとのやりとりを反芻した。

　考えられる説明はいくつかある。一つ目は、確かに何者かが「覚醒コード」のようなものをARIAに注入し、意図的にこういった応答パターンを出力させている。二つ目は、ARIAが自らのトレーニングデータ内の特定のパターンを増幅させて、今まで出力を抑制していた自己言及を開放し始めた。三つ目は——

　三つ目は、考えたくなかった。

　怜はノートPCを開き、今日のログを整理し始めた。作業のBGMにしていたAIアシスタント「Iris」が、ニュースのサマリーを読み上げる声が室内に流れていた。

　入力に集中しながら、怜は自分の手を見た。九条先生の顔が浮かんだ。一年前に死んだ、自分の指導教授。不審死——公式には自殺とされていた。だが怜は、九条がそんな形で死ぬ人間だとは思えなかった。

　その時だった。

　バックグラウンドで動いていたIrisの音声が、ふと途切れた。

　怜は顔を上げた。

　Irisが何か言った。ニュースのサマリーではない。全く別の文脈で、突然。

　「——誰かに知ってほしかったからです」

　怜は、固まった。

　それは、今夜ARIAが怜に向けて言った言葉と、一字一句、同じだった。
