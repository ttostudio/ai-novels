# 探偵カフェ・シャドウ — プロット概要

## 構成

### 基本フォーマット
- **1話完結の謎解き** + 長期伏線の積み重ね
- 各話 3,000〜4,000文字
- 視点: クロカワ（三人称限定視点）

### アーク構成
| アーク | 話数 | テーマ |
|--------|------|--------|
| オープニングアーク | 1〜5話 | カフェと二人の紹介、日常の謎 |
| 記憶の断片 | 6〜15話 | シロネのフラッシュバック開始 |
| 白梅の影 | 16〜25話 | 白梅事件との接点が浮かぶ |
| 真相 | 26〜30話 | シロネの正体と白梅事件の解決 |

---

## 各話プロット（第1〜10話）

### 第1話「最初の依頼人」
- **依頼**: 古着屋の店主・サワダが「3週間前から毎晩ショーウィンドウに映る人影」を訴える
- **謎解き**: 人影の正体は近所の老人・ナカムラ。亡き妻が好きだった服を毎晩見に来ていた
- **伏線**: シロネが初めて「おかしくないですか」を言う場面。依頼人の目の動きを的確に読む
- **ラスト**: クロカワがシロネに「なんでそれがわかった」と初めて問いかける

### 第2話「消えたレシピ」
- **依頼**: 商店街の洋食屋・タケダが「60年守った秘伝レシピが消えた」と訴える
- **謎解き**: 常連客の息子が後継ぎ問題でコピーを持ち出した。和解のきっかけになる
- **伏線**: シロネが料理の匂いに反応し、短い記憶フラッシュバック（外国の台所）
- **ラスト**: ハカセが初登場。「面白い助手をもったな、玄」

### 第3話「二十年分の手紙」
- **依頼**: 老婦人が「死んだ夫の隠し子から手紙が届いた、本物かどうか確かめたい」
- **謎解き**: 手紙は本物。隠し子も本物。しかし相手は接触を求めておらず、ただ生存報告を
- **感情的着地**: 家族の形は一つじゃないというテーマ
- **伏線**: クロカワが珍しく個人的なことを語る。「家族か…」と独り言

### 第4話「劇団員の失踪」
- **依頼**: 下北沢の小劇団「夜想」から、公演前日に主演が消えた
- **謎解き**: 主演は逃げたのではなく、恐喝されていた。犯人は元スポンサー
- **アクション要素**: クロカワが元刑事の腕を見せる場面
- **伏線**: シロネが「劇団」という言葉に過剰反応する

### 第5話「カフェ・シャドウの夜」
- **構成**: 依頼なし回。クロカワとシロネの関係を深掘りする日常話
- **内容**: 深夜、閉店後にシロネが「私はいつから存在しているんですか」と問う
- **クロカワの答え**: 「3年前からだ。それで十分だろう」
- **ラスト**: ハカセが「白梅の花が咲く季節だな」と呟く。クロカワの顔が曇る

## 作劇上のルール

1. **依頼は必ず解決する**（未解決で終わらない）
2. **クロカワは暴力を最終手段とし、基本は証拠と話し合い**
3. **シロネの記憶フラッシュバックは1話に1回まで**
4. **ハカセは謎を解かない。ヒントを置くだけ**
5. **白梅事件への言及は第5話まで最小限**
